# PLAN GRATIS Y LOCAL (SIDAM / SEEDANCE)

## 1) VERDAD CLAVE (ACTUALIZADO)

- Seedance 2.0 es oficial de ByteDance.
- En fuentes oficiales revisadas no aparece descarga publica de pesos/checkpoint para auto-hosting local.
- En documentacion oficial de Volcengine aparece que:
  - Seedance 2.0 se puede probar con cuota gratis en el centro de experiencia.
  - Aun puede no estar habilitado para llamada API directa en ese canal concreto.

Conclusion:
- Tener Seedance 2.0 100% local/offline con pesos oficiales: hoy no confirmado publicamente.
- Forma gratis oficial mas realista: cuota gratis en consola/web oficial.

## 2) LO QUE SI YA TIENES EN LOCAL

- Carpeta recuperada: `RECUPERADO_SIDAM/`
- Carpeta de alternativas open-source locales: `ALTERNATIVA_LOCAL_GRATIS/`
- Inventario automatico: `docs/INVENTARIO_LOCAL_SIDAM.md`

## 3) FORMA GRATIS (RUTA PRACTICA)

### Opcion A - Oficial (recomendada)

- Usar Seedance 2.0 en experiencia web oficial con cuota gratis.
- Ventaja: modelo real de Seedance.
- Limite: no es offline local puro.

### Opcion B - Local gratis real (alternativas abiertas)

- Modelos open-source para video en local:
  - `ALTERNATIVA_LOCAL_GRATIS/Wan2.2`
  - `ALTERNATIVA_LOCAL_GRATIS/HunyuanVideo`
  - `ALTERNATIVA_LOCAL_GRATIS/LTX-Video`
  - `ALTERNATIVA_LOCAL_GRATIS/CogVideo`
- Ventaja: 100% local posible.
- Limite: no es Seedance oficial, requiere GPU potente.
